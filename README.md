# Absolute Enable Right Click & Copy

**Universal Copy & Enable Right Click (Smart Mode)** is a Tampermonkey user script that removes copy protection, forces text selection, and unlocks the right-click menu on websites that block them.


## üöÄ Features

* **Force Text Selection:** Re-enables text highlighting on sites that disable it (using CSS injection).
* **Enable Copying:** Removes restrictions on `Ctrl+C`, `Cut`, and `Paste` events.
* **Smart Right-Click:**
    * **General Sites:** If a site blocks the context menu, hold `Alt` + `Right Click` to bypass the block and open the browser's native menu.

    * [Violentmonkey](https://violentmonkey.github.io/)

3.  Click **Install** in the tab that opens.

## üõ† Usage

* **Text Selection:** Works automatically on all sites.
* **Copying:** Use `Ctrl + C` as usual.
* **Right Click:**
    * On sites that block right-click: Hold the **`Alt`** key on your keyboard and click the Right Mouse Button.
